<h2 class="heading">Problem Statement</h2>
<p>
Start from integer <code class="inline">1</code>, remove any integer that contains <code class="inline">9</code> such as <code class="inline">9, 19, 29...</code>
</p>
<p>
So now, you will have a new integer sequence: <code class="inline">1, 2, 3, 4, 5, 6, 7, 8, 10, 11, ...</code>
</p>
<p>
Given a positive integer <code class="inline">n</code>, you need to return the <code class="inline">n-th</code> integer after removing. Note that <code class="inline">1</code> will be the first integer.
</p>

<b>Example 1:</b>
<blockquote>
<p>
<b>Input</b>: n = 9<br/>
<b>Output</b>: 10<br/>
</p>
</blockquote>
